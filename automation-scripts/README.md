## Email Automation Script

A simple Python-based learning project that sends emails automatically using the SMTP protocol. 
This script demonstrates how email automation works in Python and is created for hands-on practice and understanding automation concepts.

### Features
- Send emails automatically using Python
- Secure SMTP connection
- Customizable subject and message body
- Simple and beginner-friendly code

### Technologies Used
- Python
- smtplib
- email.message

1. Install Python (3.x)
